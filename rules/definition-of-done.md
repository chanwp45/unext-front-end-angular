# Definition of Done (DoD)

A user story or task is considered **complete** only when all items below are checked:

## Checklist

- [ ] Code written & peer-reviewed
- [ ] Unit tests pass (≥ 80% coverage)
- [ ] Manual testing completed
- [ ] Code follows SOP standards ([Coding Standards](./coding-standards.md))
- [ ] Documentation updated
- [ ] No console errors/warnings
- [ ] Accessibility (WCAG 2.1 AA) verified
- [ ] Performance checked (Lighthouse ≥ 90)
- [ ] Pull request approved
- [ ] Merged to main branch
